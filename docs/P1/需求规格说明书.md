## 1. 引言

### 1.1 目的

本文档描述 CampusHub 校园综合服务平台的功能需求和非功能需求，为开发团队提供明确的开发目标，为测试团队提供验收依据。

### 1.2 范围

CampusHub 是一个面向大学生的校园互助服务平台，支持快递代取、学习辅导、二手交易、组队匹配等功能。

### 1.3 定义

| 术语 | 定义 |
|------|------|
| 发布者 | 在平台上发布互助需求的用户 |
| 接单者 | 接受并完成互助需求的用户 |
| 信用分 | 用户信誉评价体系 |

- 本阶段用户调研报告

## 2. 总体描述

### 2.1 产品视角

CampusHub 是一个 Web 应用，采用 Vue3 + JavaScript 技术栈，为大学生提供便捷的校园互助服务。

### 2.2 用户特征

| 用户类型 | 特征描述 |
|---------|---------|
| 普通学生 | 主要使用快递代取、二手交易等功能 |
| 学霸/达人 | 提供学习辅导、技能分享服务 |
| 社团/组织 | 发布活动、招募队员 |

### 2.3 约束

- 开发周期：10 周
- 技术栈：Vue3 + JavaScript
- 预算限制：学生项目，优先使用免费服务

### 2.4 假设

- 用户拥有智能手机和校园网络
- 用户愿意进行实名认证
- 学校允许此类平台运营
