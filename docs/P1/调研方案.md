## 一、问卷设计

### 调研问卷（10-15 个问题）

**基本信息**
1. 您的年级：大一 / 大二 / 大三 / 大四 / 研究生
2. 您平均每周使用校园互助类平台的频率：
   - 每天多次
   - 每天一次
   - 每周几次
   - 很少使用
   - 从不使用

**需求场景**
3. 您最常遇到的校园生活困扰是什么？（多选）
   - 快递取件时间冲突
   - 学习资料难以获取
   - 二手物品处理困难
   - 找不到合适的队友/搭子
   - 课程作业求助无门
   - 其他：______

4. 您最需要哪类互助服务？（排序）
   - 快递代取
   - 学习辅导/答疑
   - 二手交易
   - 组队匹配（比赛/项目/运动）
   - 失物招领
   - 其他：______

5. 您愿意为一次快递代取服务支付的价格区间：
   - 1-2 元
   - 3-5 元
   - 5-10 元
   - 看距离而定

**使用习惯**
6. 您更倾向于通过什么方式发布/查找互助需求？
   - 微信群/QQ群
   - 校园论坛/BBS
   - 专门的 App/小程序
   - 朋友圈/社交平台

7. 使用互助平台时，您最担心的问题是什么？（多选）
   - 交易安全/被骗
   - 个人信息泄露
   - 服务质量无法保障
   - 响应速度慢
   - 其他：______

8. 您希望平台提供哪些保障措施？（多选）
   - 实名认证
   - 信用评分系统
   - 平台担保交易
   - 投诉举报机制
   - 其他：______

**开放性问题**
9. 请描述一次您印象深刻的互助经历（成功或失败）：

10. 如果您来设计一个校园互助平台，最希望它具备什么功能？

## 二、访谈提纲

### 用户访谈（5-8 个开放式问题）

**背景了解**
1. 请简单介绍一下您的年级和专业，平时课余时间一般怎么安排？

**痛点挖掘**
2. 在校园生活中，有没有什么事情让您觉得"要是有人能帮个忙就好了"？能具体说说吗？
3. 您现在是怎么解决这类需求的？（微信群、论坛、朋友帮忙等）这些方式有什么不方便的地方？

**场景探索**
4. 如果有一个专门的校园互助平台，您最希望在上面做什么？为什么？
5. 您会愿意在平台上帮助别人吗？什么情况下愿意，什么情况下不愿意？

**安全顾虑**
6. 使用陌生人互助服务时，您最担心什么？什么样的机制能让您放心使用？

**竞品反馈**
7. 您用过闲鱼、校园表白墙之类的平台吗？体验如何？有什么可以借鉴或避免的？

**期望收集**
8. 如果用三个词形容您理想中的校园互助平台，会是什么？为什么？
