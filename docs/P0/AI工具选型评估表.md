# AI 工具选型评估表

日期：2026.4.6

---

## 一、AI 编程助手

| 候选工具 | 核心功能 | 易用性 | 成本 | 团队熟悉度 | 综合评分 | 是否选用 | 理由 |
|---------|---------|------|------|-----------|---------|---------|------|
| GitHub Copilot | 代码补全、函数生成、注释生成 | ⭐⭐⭐⭐⭐ | 付费（学生免费）| 高 | 9/10 | 否 | 功能强大但需翻墙，国内网络不稳定 |
| Cursor | AI 编辑器、代码生成、重构 | ⭐⭐⭐⭐ | 付费 | 中 | 8/10 | 否 | 体验好但学习成本较高 |
| 通义灵码（Lingma）| 代码补全、智能提示、单元测试生成 | ⭐⭐⭐⭐⭐ | 免费 | 高 | 9/10 | **是** | 国内可用，IDE 插件成熟，免费 |
| WorkBuddy | 多模型对话、文件操作、远程控制 | ⭐⭐⭐⭐ | 免费 | 中 | 8/10 | **是** | 可接入不同大模型，工作效率高 |
| Codeium | 免费代码补全、聊天 | ⭐⭐⭐⭐ | 免费 | 低 | 7/10 | 否 | 功能较基础 |

**最终选用：** 通义灵码（Lingma）+ WorkBuddy
**选择理由：**
- 通义灵码：国内网络友好，IDE 插件成熟，代码补全准确率高，完全免费
- WorkBuddy：支持多模型切换（GPT-4、Claude、Gemini 等），适合复杂任务和远程协作

## 二、AI 对话工具

| 候选工具 | 核心功能 | 易用性 | 成本 | 团队熟悉度 | 综合评分 | 是否选用 | 理由 |
|---------|---------|------|------|-----------|---------|---------|------|
| ChatGPT | 通用对话、代码解释、文档生成 | ⭐⭐⭐⭐⭐ | 付费 | 高 | 9/10 | 否 | 需翻墙，API 调用成本高 |
| Claude | 长文本处理、代码分析 | ⭐⭐⭐⭐ | 付费 | 中 | 8.5/10 | 否 | 国内访问受限 |
| Gemini | 多模态、代码生成、搜索增强 | ⭐⭐⭐⭐ | 低价/免费 | 中 | 8/10 | **是** | 价格便宜，支持中文 |
| 文心一言 | 中文对话、文档生成 | ⭐⭐⭐ | 免费 | 高 | 7/10 | 否 | 代码能力相对较弱 |
| DeepSeek | 代码生成、数学推理 | ⭐⭐⭐ | 免费 | 低 | 6/10 | 否 | 幻觉较严重，不稳定 |

**最终选用：** Gemini
**选择理由：**
- 价格低廉（API 成本约为 GPT-4 的 1/10）
- 支持中文，代码生成能力较强
- 支持 Google 搜索增强，信息较新

## 三、AI 代码审查工具

| 候选工具 | 核心功能 | 易用性 | 成本 | 团队熟悉度 | 综合评分 | 是否选用 | 理由 |
|---------|---------|------|------|-----------|---------|---------|------|
| CodeRabbit | AI 自动 PR Review、代码分析 | ⭐⭐⭐ | 付费 | 低 | 7/10 | 否 | 第三方工具，需额外配置 |
| AI Code Review | GitHub Action 集成 | ⭐⭐⭐ | 免费/付费 | 低 | 6/10 | 否 | 配置复杂，效果一般 |
| GitHub Pull Request + Code Review | 人工 Review + CI 检查 | ⭐⭐⭐⭐⭐ | 免费 | 高 | 9/10 | **是** | 零成本，流程自然，可追溯 |
| SonarQube | 静态代码分析、质量门禁 | ⭐⭐⭐ | 免费（社区版）| 中 | 8/10 | 否 | 适合后期，初期配置较重 |

**最终选用：** GitHub Pull Request + Code Review
**选择理由：**
- 与现有 GitHub 工作流无缝集成
- 零额外成本，无需学习新工具
- 支持强制 Review 规则（至少 1 人 Approve）
- Review 记录可追溯，便于复盘

## 四、AI 测试工具

| 候选工具 | 核心功能 | 易用性 | 成本 | 团队熟悉度 | 综合评分 | 是否选用 | 理由 |
|---------|---------|------|------|-----------|---------|---------|------|
| Vitest | Vue 单元测试、组件测试 | ⭐⭐⭐⭐⭐ | 免费 | 中 | 9/10 | **是** | Vue 生态原生，极速运行 |
| Jest | JavaScript 单元测试 | ⭐⭐⭐⭐ | 免费 | 高 | 7/10 | 否 | Vue 3 支持需额外配置 |
| Cypress | E2E 端到端测试 | ⭐⭐⭐⭐ | 免费 | 中 | 8/10 | 否 | 适合后期补充，非单元测试首选 |
| Playwright | 跨浏览器 E2E 测试 | ⭐⭐⭐ | 免费 | 低 | 7/10 | 否 | 学习成本较高 |

**最终选用：** Vitest
**选择理由：**
- Vue 团队官方出品，与 Vue 3 深度集成
- 基于 esbuild，运行速度比 Jest 快 10-100 倍
- 零配置即可测试 Vue 单文件组件（SFC）
- 原生支持 TypeScript，API 兼容 Jest

## 五、总结

| 工具类型 | 选用工具 | 总成本 | 适用阶段 |
|---------|---------|--------|---------|
| AI 编程助手 | 通义灵码 + WorkBuddy | 免费 | 全程 |
| AI 对话工具 | Gemini | 低价 | 全程 |
| AI 代码审查 | GitHub PR + Code Review | 免费 | 全程 |
| AI 测试工具 | Vitest | 免费 | 开发阶段 |

**选型原则：**
1. **成本优先**：学生团队，优先选择免费/低价工具
2. **生态契合**：优先选择 Vue 生态原生工具
3. **国内可用**：避免网络问题影响开发效率
4. **渐进增强**：初期轻量级，后期按需引入 SonarQube、Cypress 等
